## 📂 Features

sublodex combines many tools into one window so you stay focused on your work.

### Multi-Project Management
Switch between different folders and projects in seconds. Each project gets its own workspace, which keeps your files clean and organized. You never need to close one project to start another.

### Live File Streaming
The application tracks changes to your files in real time. If you edit or add a document, sublodex updates its view immediately. This ensures your project stays current while you work.

### Integrated Terminal
You can run commands directly inside the app. Whether you work with files on your local hard drive or connect to a remote server through an SSH link, the terminal handles your requests in one window.

### Built-in Git Tools
Track your file history and manage changes with visual tools. You can save versions of your work, look at past updates, and combine code from different sources without typing complex commands.

### Multi-Session Support
Run several sessions for the same project. This allows you to test code in one window while you write in another. You can keep multiple workflows open to compare results or handle different tasks at the same time.

## 🧩 Frequent Questions

### Does sublodex store my files in the cloud?
No. Your files stay on your computer. This keeps them private and lets you work without an internet connection after the app is installed.

### Can I connect to my remote server?
Yes. Open the terminal inside the app and use the standard connection command for your server. The app retains your session settings for easy re-entry.

### How do I remove the app?
Open your Windows Settings, go to Apps, find sublodex, and select Uninstall. This removes the program from your system.

### Can I use this for non-coding tasks?
While sublodex focuses on software development projects, many users find it helpful for managing large sets of text documents and tracking versions of their drafts.

### What happens if I update the app?
Updating the app preserves your settings and your active sessions. You do not lose your project configuration when you install a newer version.

## 🛡️ Privacy and Safety

We believe your code belongs to you. sublodex functions as a local tool. It does not send your files or your project data to external servers. All processing happens on your own hardware, which ensures that your logic remains entirely under your control. We value your digital security above all else.
